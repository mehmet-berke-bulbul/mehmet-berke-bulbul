## 🚀 Featured Project

### Intelligent Gimbal Tracking System
**Intel RealSense + YOLO + NEMA17 + Two-Axis Gimbal**

An intelligent tracking system that detects and follows objects using computer vision, depth information and stepper motor control.

**Main components**
- Intel RealSense B415 depth camera
- YOLO object detection
- Depth-based distance estimation
- NEMA17 stepper motors
- Two-axis gimbal mechanism
- Real-time target tracking

🔗 [View the project repository](https://github.com/KULLANICI_ADIN/intel-b415-akilli-gimbal-takip-sistemi)
